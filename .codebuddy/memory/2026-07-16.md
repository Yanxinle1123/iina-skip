# 2026-07-16 工作日志

## 添加清除识别缓存 UI 功能

在偏好设置页面添加了"缓存管理"区域，支持一键清除视频指纹检测的磁盘缓存和内存缓存。

### 修改的文件
1. **`detectors/video-match.js`** - 添加 `clearVideoMatchCache()` 函数，删除 `$TMPDIR/iina-skip-cache/` 目录 + 清空 `resultCache` Map
2. **`preferences.html`** - 添加"缓存管理"区域 + "清除识别缓存"按钮 + 状态反馈
3. **`main.js`** - 添加 `clearVideoMatchCache` 引用 + 每 2 秒轮询 `clear_video_cache_trigger` sentinel 偏好
4. **`Info.json`** - 在 `preferenceDefaults` 中添加 `clear_video_cache_trigger: ""`

### 通信流程
1. 用户点击按钮 → `preferences.html` 设置 `clear_video_cache_trigger = Date.now()`
2. `main.js` 每 2 秒轮询该偏好 → 检测到变化 → 调用 `clearVideoMatchCache()`
3. `clearVideoMatchCache()` 执行 `rm -rf $TMPDIR/iina-skip-cache` + `resultCache.clear()`
4. 日志输出清除结果
